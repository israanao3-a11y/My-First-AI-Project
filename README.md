# 🚀 Projet de Prédiction des Notes (IA)

Ce projet est ma première application en **Intelligence Artificielle** utilisant Python. Il permet de prédire la note finale d'un étudiant en fonction de ses heures d'étude.

## 📊 Fonctionnalités
- Visualisation des données avec **Matplotlib**.
- Modèle de **Régression Linéaire** simple avec **Numpy**.
- Prédiction automatique des résultats futurs.

## 🛠️ Outils utilisés
- **Langage :** Python 🐍
- **Bibliothèques :** Numpy, Matplotlib.
- **Environnement :** Jupyter Notebook (VS Code).

## 📈 Résultat
L'IA trace une ligne de prédiction (en rouge) qui montre la corrélation entre l'étude et la réussite.
